Welcome to AI Assisted Revenue Forecasting

Your tool for machine learning-based revenue forecasting in FPB Finance / OP

Instructions

0. Prerequisites

  • Saving and loading forecast adjustments locally requires a chromium-based browser (e.g., Chrome, Edge).
  • Install the revenue forecasting application (see GitHub for Windows instructions).
  • Export revenue data from the finance system. For support, contact Joonas Rosi.
  • Ensure data is in CSV format. If using Excel, save it as a CSV file.
  • Timestamps must follow dd/mm/yyyy HH:MM format. Be cautious when using Excel.
  • Ensure the calendar file is up-to-date. It should include:
    • Monthly working days per location
    • Historical rate raise percentages

1. Upload Data

  • Copy data to an accessible directory for the web UI.

2. Start Application

  • Navigate to src: cd <application root>/src.
  • Run: flask --app app run.
  • Open http://127.0.0.1:5000 in your browser. You should see the tabs: Home, Preprocessing, and Forecast.

3. Preprocess Data

  • Go to the Preprocess tab.
  • Select the revenue data and calendar files.
  • Click Process Data.
  • Locate the output file data.csv in your run directory.
  • Manually remove incomplete monthly data if necessary (e.g., partial January 2025 revenue).

4. Forecast

  • Go to the Forecast tab.
  • Select models, add corrections, and additional revenue components.
  • Click Calculate Totals to update adjustments.
  • Click Save Changes to preserve inputs for future runs.
  • Export the forecast as a CSV file for Excel.
